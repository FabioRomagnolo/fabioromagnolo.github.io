  • Background Matting

    The Image Matting allows to replace the background considering the transparency, so it is one of the most accurate background replacement techniques.
    The Background Matting simplify this operation by taking as input the background you want to remove.
    In this demo you are able to try different Neural Networks with various parameters, trained for this task.

    NOTE: there is no server running the model, it will be on your device!

    L' Image Matting permette di sostituire lo sfondo considerando la trasparenza, il che la rende una delle migliori tecniche di rimozione dello sfondo.
    Il Background Matting semplifica tale operazion prendendo come input lo sfondo da eliminare.
    In questa demo avrai la possibilità di testare più reti neurali con diversi parametri, addestrate a tale scopo.

    NOTA: nessun server esegue i modelli, ma lo farà il tuo dispositivo!

  • Guitar Chords Recognition

    A Computer Vision pipeline consisting of the following steps: hands detection, strings detection for image alignment, chords recognition by MobileNetV2 neural network pre-trained on ImageNet and fine-tuned on hand-made chords dataset.

    Una pipeline di Computer Vision composta dai seguenti step: rilevazione delle mani, rilevazione delle corde per l'allineamento dell'immagine, riconoscimento degli accordi tramite la rete neurale MobileNetV2 pre-addestrata su ImageNet e perfezionata su dataset costruito ad-hoc.
